# Icons Directory

This directory should contain the following icon files for the Chrome extension:

- `icon16.png` - 16x16 icon for the extensions toolbar
- `icon48.png` - 48x48 icon for the extensions management page  
- `icon128.png` - 128x128 primary icon for installation

## Icon Requirements

- Format: PNG
- Sizes: 16x16, 48x48, 128x128 pixels
- Background: Transparent or solid color
- Style: Consistent with the extension's branding

## Placeholder

For development purposes, you can create simple placeholder icons or use any 16x16, 48x48, and 128x128 PNG files with the correct names.
